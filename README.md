# リアルタイム音声字幕＆音の見える化

マイクに入力された日本語をリアルタイムで大きな文字に変換し、同時に音の波形・周波数スペクトル・音量を表示する授業用Web教材です。

## 概要

この教材では、次の2つを同時に観察できます。

1. マイクから入力された音声が、日本語の文字へ変換される様子
2. 同じ音声が、波形・周波数・音量のデータとして表される様子

音声認識の結果と音の可視化を並べて表示することで、「声がデジタルデータとして処理される仕組み」を学習しやすくすることを目的としています。

## 主な機能

- 日本語音声のリアルタイム字幕表示
- 認識中の内容を大きな文字で表示
- 確定した字幕の蓄積
- 直近5件の発話履歴
- 音声波形のリアルタイム表示
- 周波数スペクトルのリアルタイム表示
- 音量メーター
- 字幕文字サイズの変更
- 字幕の全画面表示
- グラフ表示の一時停止
- 経過時間・文字数・確定文数の表示

## 想定する利用場面

- 中学校技術科「情報の技術」の授業
- 音声認識の仕組みを説明する授業
- アナログ音声とデジタルデータの関係を扱う授業
- マイク、波形、周波数、音量の観察
- 生成AIとの音声対話を支える技術の説明
- 音声情報を文字で提示する仕組みの体験

## 動作環境

音声認識機能の互換性を考慮し、最新版の次のブラウザを推奨します。

- Google Chrome
- Microsoft Edge

利用には、次の条件が必要です。

- マイクを使用できる端末
- ブラウザでのマイク利用許可
- HTTPSで公開されたWebページ
- 音声認識サービスを利用できるインターネット環境

FirefoxやSafariなどでは、音声認識機能が動作しない場合があります。

## 使い方

1. 「音声認識を開始」を押します。
2. ブラウザから求められた場合は、マイクの使用を許可します。
3. マイクへ自然な速さで話します。
4. 画面に表示される字幕、波形、周波数、音量を観察します。
5. 終了するときは「音声認識を停止」を押します。

## GitHub Pagesでの公開方法

1. リポジトリの `Settings` を開きます。
2. 左側のメニューから `Pages` を選びます。
3. `Build and deployment` の `Source` を `Deploy from a branch` にします。
4. Branchを `main`、フォルダーを `/(root)` に設定します。
5. `Save` を押します。

公開URLは通常、次の形式になります。

```text
https://ユーザー名.github.io/realtime-speech-caption-visualizer/
```

## 技術構成

外部ライブラリやビルドツールは使用していません。

- HTML
- CSS
- JavaScript
- Web Speech API
- Web Audio API
- Canvas API
- MediaDevices API

`index.html` 単体で動作します。

## 音声データとプライバシー

波形・周波数・音量の可視化は、利用者のブラウザ内で処理されます。

一方、音声から文字への変換には、ブラウザまたはOSが提供する音声認識サービスを利用します。そのため、利用環境によっては、音声データが外部サーバーで処理される場合があります。

授業で利用するときは、次の情報を発話しないよう指導してください。

- 氏名
- 住所
- 電話番号
- メールアドレス
- 個人を特定できる情報
- その他の機密情報

本教材は、入力した音声や認識結果を、独自のサーバーやデータベースへ保存する機能を持っていません。

## 教材としての注意

画面に表示される波形や周波数スペクトルは、同じマイク入力を可視化したものです。ただし、表示された波形やスペクトルを、この教材のJavaScriptが直接解析して日本語を判定しているわけではありません。

日本語への変換は、ブラウザまたはOSの音声認識サービスが行っています。

音声認識結果には誤りが含まれる場合があります。正確な記録、成績評価、本人確認、医療、防災などの重要な判断には使用しないでください。
